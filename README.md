# Express JS Boilerplate

An basic express boilerplate for APIs

## Requirements

- Node 16.17 LTS

## Dependencies

- Express JS
- Dotenv
- prisma

## Commands

### Installation

```bash
 npm install
 yarn intall
```

### Run

```bash
npm run build ## compile source
npm run start ## start compiled source on dis/ folder
npm run dev ## watch mode

yarn build
yarn start
yarn dev
```
